# 射的ゲーム

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 射的ゲーム

## ゲームの遊び方
* spaceキーでゲームを開始する
* マウスでカーソルを操作し、クリックで射撃
* 的に当たればスコアが加算される（的：10点　ハチ：30点）
![title](fig/ゲーム画面.png)
* 時間内60秒でのスコアで競う
* リトライする場合は　Rキー

## ゲームの実装
### 共通基本機能
* ゲームが開始されると的が表示される
* 的をクリックすると、スコアが加算される

### 分担追加機能
* 背景画像・エフェクト・動く的担当（冨永）：
* ゲーム開始、終了画面担当（矢島）：
ゲーム開始画面を表示し、spaceキーを押すことでゲームが開始される。
ゲームが終了するとリザルト画面が表示される。Rキーを押すことでリトライできる。　
* 的の出現方法（家崎）：
複数の的の出現・球が当たったら消えるように・時間経過で的が変わるように
* 制限時間・スコア管理（髙橋）：制限時間のタイマーと時間が少なくなった時に色が変わって強調されるようにした。
